Einstellungen

Alles, was im Einstellungs-Dialog konfigurierbar ist.

Speicherort

Einstellungen werden in %APPDATA%\SSHWinManager\config.json gespeichert. Verbindungen und Benutzerprofile liegen in einer separaten Datenbank im selben Ordner.

Allgemein

Einstellung Standard Effekt
Mit Windows starten aus App startet automatisch beim Windows-Login.
In Tray minimieren an Schließen minimiert die App ins System-Tray statt sie zu beenden.
Auto-Login mit Windows-Account aus Login wird übersprungen, wenn Windows-Benutzer mit App-Profil übereinstimmt.
Auto-Reconnect aus Unterbrochene Laufwerke werden automatisch neu verbunden.
Laufwerke beim Start wiederherstellen an Beim App-Start werden zuletzt aktive Laufwerke automatisch eingehängt.
Check-Intervall 30 s Wie oft der Verbindungsstatus geprüft wird.
Debug-Modus aus Aktiviert ausführliches Logging und ein Live-Log-Fenster.
PuTTY verwenden aus PuTTY statt OpenSSH für SSH-Sessions verwenden.
PuTTY-Pfad Standard-Pfad Pfad zur PuTTY-Installation.

Design wechseln

Im Einstellungs-Dialog → Erscheinungsbild → Hell oder Dunkel. Pro Benutzer gespeichert, wirkt sofort.

Sprache wechseln

Einstellungen → Sprache. Verfügbar: Deutsch und Englisch. Die Oberfläche wechselt sofort.

Debug-Modus

Aktiviert detailliertes Logging in %APPDATA%\SSHWinManager\app.log und ein Live-Log-Fenster. Hilfreich bei Verbindungsproblemen.
