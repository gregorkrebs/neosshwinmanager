Download.

NEO SSH-Win Manager ist kostenlos und Open Source. Einfach herunterladen und starten — keine Installation.

Empfohlen

NeoSSHWinManager.exe

Die Hauptanwendung mit allem: Verbindungsverwaltung, Tray, mehrere Benutzerprofile, Live-Systemstatus.

~25 MB · Windows 10 / 11 (x64)

GUI herunterladen
Optional

NeoSSHWinManager-cli.exe

Konsolen-Companion für Skripte und Automatisierungen. Nutzt die Verbindungen der laufenden Hauptanwendung.

~10 MB · erfordert laufende GUI

CLI herunterladen
Windows SmartScreen Die App ist nicht code-signiert (Open-Source-Projekt). Windows zeigt beim ersten Start möglicherweise eine Warnung. „Weitere Informationen" → Trotzdem ausführen klicken. Lade die App ausschließlich aus dem offiziellen GitHub-Releases herunter.

Voraussetzungen

Zwei kostenlose Programme werden benötigt.

Diese erledigen das SSH-Mounting im Hintergrund — NEO SSH-Win Manager ist die Bedienoberfläche.

Programm Wozu Download
WinFsp Stellt das Dateisystem-Interface für Windows bereit. winfsp/winfsp
SSHFS-Win Verbindet SSH mit dem Dateisystem. winfsp/sshfs-win
Windows 10 / 11 x64. Andere Versionen werden nicht getestet. —
OpenSSH Optional, für die eingebaute Terminal-Funktion. Windows-Feature (vorinstalliert)

Installationsreihenfolge

  1. WinFsp installieren.
  2. SSHFS-Win installieren.
  3. NeoSSHWinManager.exe herunterladen — keine Installation nötig, einfach starten.
  4. App starten, Benutzerprofil anlegen, erste Verbindung hinzufügen.

Aus Quellcode bauen

Selbst kompilieren.

Mit Python 3.11+ und PyInstaller.

git clone https://github.com/gregorkrebs/neosshwinmanager.git
cd neosshwinmanager

python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt

# Direkt aus Quellcode starten
python main.py

# Beide EXE-Dateien bauen (GUI + CLI)
.\build_dual.ps1

EXE-Dateien werden im dist/-Ordner gespeichert.
